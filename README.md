# 🎁 OwnRewards 2.0

> **The Ultimate Enterprise Loyalty & Retention SaaS Engine.**

[![Live Demo](https://img.shields.io/badge/Live_Demo-Vercel-black?style=for-the-badge&logo=vercel)](https://own-rewards2-0.vercel.app)
[![Tech Stack](https://img.shields.io/badge/Built_with-React_&_Vite-6F4DE6?style=for-the-badge&logo=react)](https://react.dev)

**OwnRewards** is a premium, enterprise-grade Customer Loyalty and Retention platform designed to help retail brands, restaurants, and e-commerce stores scale their customer base and increase repeat purchases. By ditching the traditional aggregator model, OwnRewards allows businesses to completely own their customer relationships with zero transaction fees.

---

## ✨ Key Features

- **🏆 Tiered Loyalty Systems:** Create highly customizable VIP programs (e.g., Bronze, Silver, Gold, Platinum) based on customer spend and engagement.
- **⚡ Advanced Rule Engine:** Leverage 30+ custom event triggers and 10+ action types. Set complex rules like *"If a customer spends ₹500 on a Tuesday, grant 2x points."*
- **📈 Smart Marketing & Referrals:** Dual-sided referral programs, A/B testing for campaigns, and a powerful smart coupon generator (including advanced BOGO workflows).
- **💬 Omnichannel Integrations:** Native POS integrations (like Petpooja) and deep WhatsApp connectivity (via **ownChat**) to instantly notify customers about points and expiring rewards.
- **📊 AI-Powered Analytics:** Track Cohort Retention, Churn Risk, Predicted LTV, and precise ROI metrics per reward.
- **🎨 Premium UI/UX:** Built with a $100M+ enterprise SaaS aesthetic, featuring smooth micro-animations, glassmorphism, dynamic dark mode, and optimized responsive layouts.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Make sure you have Node.js and npm installed on your machine.

### Installation

1. **Clone the repo**
   ```sh
   git clone https://github.com/Vignesh-058/OwnRewards2.0.git
   ```
2. **Navigate to the project directory**
   ```sh
   cd OwnRewards2.0
   ```
3. **Install NPM packages**
   ```sh
   npm install
   ```
4. **Run the development server**
   ```sh
   npm run dev
   ```
5. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.
